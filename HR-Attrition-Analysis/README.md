# HR Attrition Analysis

## Overview
Analysis of IBM HR dataset (1,470 employees) to identify key drivers of employee attrition.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- MySQL
- Power BI (DAX, Power Query)

## Key Findings
- Overall attrition rate: 16.12%
- Overtime workers leave at 30.53% vs 10.44% for non-overtime
- Sales Representatives have 39.76% attrition — highest by role
- Employees who left earned $2,045 less per month on average

## Files
- `hr_analysis.py` — Python EDA and visualizations
- `load_hr_data.py` — MySQL data loading script
- `HR_Dashboard.pdf` — Power BI dashboard export
